Demo Scenario

학기말 최종 시연에서 보여드릴 주요 기능들

시연 시나리오

1. 팀 브레인스토밍 세션 시작

00:00-00:30

팀원 3명이 동시에 접속하여 새로운 프로젝트 아이디어를 브레인스토밍하는 상황을 시연합니다.

2. 자유로운 아이디어 배치

00:30-01:30

각자 다른 위치에 아이디어를 자유롭게 배치하고, 드래그 앤 드롭으로 위치를 조정하는 과정을 보여줍니다.

3. 실시간 협업 동기화

01:30-02:30

한 사용자가 아이디어를 추가하거나 수정할 때 다른 사용자들의 화면에 실시간으로 반영되는 과정과, 채팅과 음성 통화를 통한 팀원 간 소통을 시연합니다.

4. AI 클러스터링 결과

02:30-03:00

입력된 아이디어들이 AI에 의해 자동으로 그룹화되고, 관련성 있는 아이디어들이 클러스터로 묶이는 과정을 보여줍니다.

시연 포인트

다중 사용자

3명이 동시 접속

실시간 동기화

지연 시간 < 100ms

AI 분석

자동 클러스터링

기술적 하이라이트

  • WebSocket 기반 실시간 통신
  • 무한 캔버스 렌더링
  • CRDT 충돌 해결
  • NLP 기반 클러스터링

예상 결과

입력: 15개의 다양한 아이디어
출력: 4개의 의미적 클러스터
시간: 3분 내 완료

시연 목표: 기존 도구들과의 차별점을 명확히 보여주고, 실시간 협업과 AI 기반 아이디어 분석의 실용성을 입증하여 1ONmusk의 혁신성을 전달합니다.
