API와 REST API 차이

API는 넓은 개념이고, REST API는 웹에서 API를 설계하는 대표적인 스타일입니다.

구분 API REST API
의미 프로그램끼리 소통하기 위한 약속 전체 HTTP 주소와 메서드를 자원 중심으로 사용하는 API 설계 방식
범위 더 넓음. 웹 API, 라이브러리 API, 운영체제 API 등이 모두 포함됨 웹 서버와 클라이언트가 데이터를 주고받을 때 많이 사용함
예시 navigator.geolocation, 카카오 지도 SDK, 서버 API GET /api/posts, POST /api/posts, DELETE /api/posts/1
핵심 어떻게 기능을 호출할지 정한 약속 자원을 URL로 표현하고 HTTP 메서드로 행동을 표현
수업에서는 "React가 서버 API를 호출한다"를 먼저 이해하고, 그 API를 REST 방식으로 깔끔하게 설계하면 어떤 장점이 있는지 연결해서 설명하면 좋습니다.

HTTP 메서드

메서드 뜻 예시
GET 데이터를 가져올 때 사용 GET /api/posts
POST 새 데이터를 만들 때 사용 POST /api/posts
PATCH 기존 데이터의 일부를 수정할 때 사용 PATCH /api/posts/1
PUT 기존 데이터를 통째로 바꿀 때 사용 PUT /api/posts/1
DELETE 데이터를 삭제할 때 사용 DELETE /api/posts/1

실습 서버 API

api-server/server.js를 실행하면 인스타그램 스타일 게시물 데이터를 반환하는 서버가 켜집니다.

서버 실행

cd api-server
npm start

이번 강의에서 쓰는 요청

GET /api/posts

화면에 게시물 목록을 보여주려면 서버에 있는 데이터를 가져와야 합니다. 그래서 새로 만들거나 수정하는 요청이 아니라 GET 요청을 사용합니다.

왜 이 주소를 쓰나요?

/api/posts는 "게시물 목록"이라는 데이터를 가리키는 주소입니다. REST API에서는 주소가 데이터를 나타내고, 메서드가 행동을 나타냅니다.

{
  "count": 4,
  "data": [
    {
      "id": 1,
      "author": {
        "name": "민서",
        "username": "minseo_daily"
      },
      "imageUrl": "https://picsum.photos/seed/cafe-window/900/900",
      "caption": "과제 끝내고 마시는 커피가 제일 맛있다.",
      "likes": 128,
      "liked": false,
      "tags": ["cafe", "study", "daily"]
    }
  ]
}

서버 공개 주소 연결

백엔드 파트에서 API 서버를 열고, 터널 서비스로 외부에서 접속 가능한 주소를 만듭니다. 프론트엔드는 그 주소로 API를 요청합니다.

같은 와이파이

fetch("http://192.168.0.69:4000/api/posts")

백엔드 서버와 프론트엔드 개발 환경이 같은 네트워크에 있을 때 사용할 수 있습니다.

터널 서비스

fetch("https://quiet-plants-spend.loca.lt/api/posts")

와이파이가 다르거나 외부 접속이 필요할 때 사용할 수 있습니다.

React 코드에서 사용할 주소

const API_URL = "https://quiet-plants-spend.loca.lt";

fetch(`${API_URL}/api/posts`);
수업 중에는 터널 서비스를 켜 둔 터미널을 닫으면 공개 주소도 같이 끊어집니다.

src/api/posts.js

export async function getPosts() {
  const API_URL = "https://quiet-plants-spend.loca.lt";
  const response = await fetch(`${API_URL}/api/posts`);
  const result = await response.json();

  return result.data;
}

src/components/PostCard.jsx

export default function PostCard({ post }) {
  return (
    <div>
      <h2>{post.author.name}</h2>
      <p>@{post.author.username}</p>

      <img src={post.imageUrl} alt={post.caption} width="300" />

      <p>{post.caption}</p>
      <p>좋아요 {post.likes}개</p>
    </div>
  );
}

src/App.jsx

import { useEffect, useState } from "react";
import { getPosts } from "./api/posts";
import PostCard from "./components/PostCard";

export default function App() {
  const [posts, setPosts] = useState([]);

  useEffect(() => {
    getPosts().then((data) => {
      setPosts(data);
    });
  }, []);

  return (
    <div>
      <h1>SNS Feed</h1>

      {posts.map((post) => (
        <PostCard key={post.id} post={post} />
      ))}
    </div>
  );
}
