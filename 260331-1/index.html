Feature 01
WebRTC

다자간 실시간
음성 대화

팀원들이 장소에 구애받지 않고 동시에 접속해 음성으로 회의하고 아이디어를 나눌 수 있는 소통의 기반이자 ON-IT의 핵심 입력창(Input)입니다.

  • WebRTC 기반 저지연 다자간 실시간 음성 통신
  • Socket.IO 시그널링으로 안정적 연결 유지
  • 방(room) 단위 회의 관리 및 참여자 제어
  • 음성 입력이 AI 파이프라인으로 실시간 전달

은서

  • WebRTC 클라이언트 연결
Feature 02
Cross Platform

멀티 플랫폼
크로스 지원

Web은 구현 완료, iOS는 이번 프로젝트에서 확정 구현합니다. Windows · macOS 데스크톱 앱과 Android는 이후 단계에서 확장할 예정입니다.

  • Web — 구현 완료
  • iOS — 이번 캡스톤에서 확정 구현
  • Windows · macOS · Android — 추후 확장 예정
  • 플랫폼별 UI 최적화 및 모바일 대응 API 설계

담당 파트

Backend 2
현준
  • 모바일 대응 API 설계
  • 인증 / 네트워크 최적화
Frontend 2
민재
  • 모바일 대응 UI
  • 앱 변환 및 빌드
Feature 03
AI · PRD

자동 PRD 작성 &
프로토타이핑

음성 대화를 AI가 실시간으로 분석해 제품 요구사항 정의서(PRD)를 자동 작성하고, 완성된 PRD를 바탕으로 앱 프로토타입을 구성해 클릭 한 번으로 배포까지 이어집니다.

  • STT 파이프라인으로 음성을 텍스트로 실시간 변환
  • AI가 아이디어를 분석해 PRD 자동 생성
  • PRD 기반 앱 프로토타입 뼈대 자동 구성
  • 구글 플레이 / 앱스토어 원클릭 배포 자동화

담당 파트

Backend 3
수빈
  • 아이디어 → PRD 변환 로직
  • AI 연동
Frontend 3
영서
  • PRD 결과 화면
  • 프로토타입 UI
