🧩 조각조각 프로젝트 소개

'조각조각'은 노인과 보호자가 함께 소중한 추억을 기록하고 관리할 수 있는 플랫폼입니다. 이번 프로젝트에서 저는 프론트엔드 개발을 담당하여 사용자 친화적인 인터페이스를 구현했습니다. React, TypeScript, Vite, Tailwind CSS 등 최신 기술 스택을 활용한 개발 경험을 공유하고자 합니다.

🎯 프로젝트 개요

조각조각은 치매와 인지 저하로 소중한 기억을 잃어가는 어르신들과 그들의 가족을 위해 개발된 PWA(Progressive Web App) 서비스입니다. 이 서비스는 어르신의 추억을 보존하는 역할을 하며 가족들이 함께 어르신을 보호할 수 있도록 돕는 것을 목표로 합니다.

💡 개발 동기

고령화 사회로 접어들면서 치매 환자 수는 꾸준히 증가하고 있으며 65세 이상 노인 10명 중 1명이 치매를 겪고 있다는 통계가 있을 정도로 심각한 사회적 문제로 자리잡고 있습니다. 또한 치매는 가족들에게도 정서적·육체적 부담을 주게 됩니다. 이러한 문제 속에서 저희는 어르신의 소중한 추억을 기록하고 보존하며 가족들이 함께 어르신을 보호할 수 있도록 서비스를 만들고자 하였습니다.

🔧 개발 과정의 도전과 해결

조각조각 프로젝트를 개발하면서 겪은 다양한 도전과 그 해결 과정을 공유합니다.

조각조각 팀의 열정적인 개발 과정

1. 반응형 디자인 구현

🎯 도전 과제

노인 사용자들을 고려하여 큰 화면과 작은 화면 모두에서 사용하기 편한 인터페이스를 구현해야 했습니다.

해결 방법: Tailwind CSS의 반응형 유틸리티 클래스를 활용하여 모바일 퍼스트 접근 방식으로 디자인했습니다.

// 반응형 디자인 예시 <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4"> <div className="p-4 bg-white rounded-lg shadow-md"> // 카드 내용 </div> </div>

2. AI 모듈 통합

Google Gemini AI를 활용한 감정 분석과 인물 인식 기능을 개발했습니다. AI API 호출 최적화와 사용자 친화적인 인터페이스 설계에 중점을 두었습니다.

// AI 감정 분석 함수 예시 const analyzeEmotion = async (diaryText: string) => { try { const response = await fetch('/api/ai/emotion', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ text: diaryText }) }); const result = await response.json(); return result.emotion; } catch (error) { console.error('감정 분석 오류:', error); } };

3. 상태 관리

복잡한 상태 관리를 위해 React의 Context API와 useReducer를 활용했습니다. 전역 상태와 지역 상태를 적절히 분리하여 관리했습니다.

// Context API를 활용한 상태 관리 const AppContext = createContext(null); const AppProvider = ({ children }) => { const [state, dispatch] = useReducer(appReducer, initialState); return ( <AppContext.Provider value={{ state, dispatch }}> {children} </AppContext.Provider> ); };

👥 팀 협업의 중요성

프론트엔드와 백엔드, AI 모듈이 함께 작업하는 팀 프로젝트에서 API 설계와 데이터 구조에 대한 소통이 얼마나 중요한지 배웠습니다. 특히 실시간 기능 구현을 위해 WebSocket 통신 방식을 백엔드 팀과 협의하여 설계했습니다.

🤝 팀 협업의 가치

🤖 AI 개발 경험

Google Gemini AI를 활용한 감정 분석과 인물 인식 기능을 개발하면서 AI 모듈의 통합과 최적화에 대해 배웠습니다. 특히 노인 사용자들을 위한 직관적인 AI 인터페이스 설계의 중요성을 깨달았습니다.
