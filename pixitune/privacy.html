개인정보처리방침

PixiTune은 사용자가 음악 상태를 친구와 공유하고, 픽셀 캐릭터와 룸을 이용해 음악 취향을 표현할 수 있도록 서비스를 제공합니다. 이 문서는 PixiTune이 어떤 정보를 수집하고, 왜 사용하며, 어떻게 보호하는지 설명합니다.

시행일: 배포일 기준으로 입력

수집하는 정보

PixiTune은 서비스 제공을 위해 다음 정보를 처리할 수 있습니다.

  • 계정 정보: Apple 로그인으로 생성되는 사용자 식별자, 표시 이름, 친구 코드
  • 프로필 정보: 프로필 음악, 캐릭터 스타일, 앱 언어, 선호 재생 앱
  • 친구 및 소셜 정보: 친구 목록, 친구 요청, 즐겨찾기, 룸 초대 및 참가 정보
  • 음악 상태 정보: 사용자가 직접 저장한 상태 노래, 프로필 음악, 최근 음악 상태
  • 리워드 정보: LP 보유량, 출석 보상, 광고 보상, 미션 보상, 구매한 캐릭터 아이템
  • 알림 정보: 푸시 알림 발송을 위한 기기 토큰, 알림 설정 상태
  • 서비스 운영 정보: 오류 확인, 보안, 부정 사용 방지, 앱 설정을 위한 기본 로그 및 Firebase 관련 데이터

정보를 사용하는 목적

  • 계정 생성 및 로그인 유지
  • 친구 검색, 친구 요청, 친구 목록 표시
  • 프로필 음악과 상태 노래 표시
  • 픽셀 캐릭터, 룸, 초대 코드 기능 제공
  • LP 보상, 미션, 캐릭터 아이템 기능 운영
  • 푸시 알림 발송
  • 앱 안정성 개선, 오류 해결, 보안 유지

제3자 서비스

PixiTune은 다음 외부 서비스를 사용할 수 있습니다.

  • Firebase: 인증, Firestore 데이터베이스, Cloud Functions, Cloud Messaging, Remote Config, Analytics
  • AdMob: 광고 표시 및 보상형 광고
  • Apple 로그인: 계정 인증
  • 음악 서비스 앱 또는 웹 링크: 사용자가 선택한 음악 앱에서 노래를 열기 위한 링크 이동

음악 서비스의 계정, 재생 가능 여부, 지역 제한, 외부 앱 동작은 각 서비스의 정책을 따릅니다.

보관 및 삭제

사용자 정보는 서비스 제공에 필요한 기간 동안 보관됩니다. 사용자가 앱 내에서 계정 삭제를 요청하면, 관련 계정 데이터는 서비스 운영상 필요한 예외를 제외하고 삭제 처리됩니다.

푸시 알림

사용자가 알림을 허용한 경우 친구 요청, 음악 추천, 룸 초대, Pipo 알림 등을 받을 수 있습니다. 알림은 기기 설정 또는 앱 설정에서 변경할 수 있습니다.

아동 개인정보

PixiTune은 관련 법령에서 요구하는 경우 아동 개인정보 보호 기준을 따릅니다. 보호자가 개인정보 관련 문의를 할 경우 아래 문의처로 연락할 수 있습니다.
