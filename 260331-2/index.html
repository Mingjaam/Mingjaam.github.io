01 — Skills
재사용

반복 작업을
명령어 하나로

Skills는 자주 쓰는 워크플로를 슬래시 명령어로 만들어 저장하는 기능입니다. 팀 전체가 동일한 절차를 공유하고 한 줄로 실행합니다.

  • /commit, /review, /test 등 커스텀 명령어 생성
  • 마크다운 파일 하나로 정의, 즉시 활성화
  • 팀 레포에 공유하면 전체 표준화 가능
  • 인자(args) 전달로 유연하게 사용

실행 흐름

사용자가 /commit 입력
↓
.claude/skills/commit.md 로드
↓
git diff 분석 → 메시지 생성
↓
커밋 완료
02 — Multi-agent
병렬 처리

한 에이전트가
여러 에이전트를 지휘

Orchestrator가 작업을 분해하고 Subagent들에게 위임합니다. 독립적인 작업은 병렬로 실행되어 속도를 높이고, 컨텍스트 오염 없이 각 에이전트가 집중합니다.

  • 탐색·구현·테스트를 동시에 진행
  • 각 에이전트는 독립된 컨텍스트 창 유지
  • 결과를 취합해 최종 응답 생성
  • 복잡한 코드베이스 리뷰·마이그레이션에 효과적

오케스트레이션 구조

Orchestrator
탐색 Agent
파일 분석
구현 Agent
코드 작성
테스트 Agent
검증 실행
결과 취합
03 — Remote Control
멀티 디바이스

PC에서 시작하고
폰에서 이어서

로컬 컴퓨터에서 실행 중인 Claude Code 세션을 URL 또는 QR코드로 공유합니다. 스마트폰이나 다른 기기의 브라우저에서 접속하면 동일한 세션이 그대로 이어집니다.

  • 로컬 파일시스템, MCP, 프로젝트 설정 모두 유지
  • 네트워크 끊김 후 자동 재연결
  • 터미널·브라우저·폰에서 동시 메시지 전송
  • Pro 이상 구독 + claude.ai 로그인 필요

사용 방법

claude remote-control
↓
URL + QR코드 표시
↓
폰 카메라로 QR 스캔
↓
브라우저에서 동일 세션 연결

또는 기존 세션에서 /remote-control 입력

04 — Computer Use
UI 자동화

Claude가 직접
컴퓨터를 조작

Claude의 Dispatch 기능으로 Claude가 직접 컴퓨터를 조작합니다. Claw는 오픈소스 프로젝트인 OpenClaw와 동일한 방식으로 작동하며, 실제 UI를 클릭하고 브라우저를 제어해 작업을 완수합니다.

  • 브라우저 탐색, 양식 입력, 버튼 클릭 자동화
  • 스크린샷으로 현재 화면 상태 인식
  • 복잡한 멀티스텝 작업 자율 수행
  • 테스트 실행, 배포, 모니터링에 활용

예시 — 웹사이트 자동 테스트

"로그인 ~ 결제까지 테스트해줘"
↓
브라우저 열기 → URL 입력
↓
이메일·비밀번호 입력 → 로그인 클릭
↓
상품 선택 → 장바구니 → 결제 진행
↓
각 단계 스크린샷 저장 · 오류 감지
↓
테스트 결과 리포트 자동 생성
